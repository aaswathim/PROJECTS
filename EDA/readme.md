# 🍽️ Restaurant Preference and Rating Analysis

## 📌 Project Overview

This project analyzes **user food preferences and restaurant ratings** based on demographic and behavioral data.
The dataset contains information about users such as **age, gender, marital status, lifestyle habits, cuisine preferences, and ratings for food and service**.

The goal of this project is to explore how **user characteristics influence restaurant preferences and overall satisfaction ratings**.

---

## 🎯 Objectives

* Analyze user demographics and food preferences
* Study how lifestyle factors (alcohol, smoking, activity) affect restaurant ratings
* Identify popular cuisines among users
* Understand patterns in **food rating, service rating, and overall rating**

---

## 📂 Dataset Description

The dataset contains **200 user records** with multiple attributes.

### Important Features

| Column         | Description                     |
| -------------- | ------------------------------- |
| User ID        | Unique identifier for each user |
| Area code      | Location area code              |
| Location       | User location                   |
| Gender         | Male / Female                   |
| YOB            | Year of Birth                   |
| Marital Status | Single, Married, Divorced       |
| Activity       | Profession or activity          |
| Budget         | User spending budget            |
| Cuisines       | Preferred cuisine               |
| Alcohol        | Alcohol consumption habit       |
| Smoker         | Smoking habit                   |
| Food Rating    | Rating given for food           |
| Service Rating | Rating given for service        |
| Overall Rating | Combined satisfaction rating    |
| Often A S      | Whether user often eats out     |
