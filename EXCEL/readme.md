# 🚦 Road Accident Analysis Dashboard

## 📌 Project Overview

This project analyzes **road accident data** to understand the factors that contribute to traffic accidents and their severity.

The dataset contains detailed information about accidents such as **date, location, road conditions, weather conditions, vehicle type, and number of casualties**.

The goal of this project is to identify patterns and trends that can help improve **road safety and accident prevention strategies**.

---

# 🎯 Project Objectives

* Analyze **accident severity distribution**
* Study the effect of **weather and road conditions on accidents**
* Identify **high-risk locations**
* Analyze accidents based on **vehicle type**
* Examine the relationship between **number of vehicles and casualties**

---

# 📂 Dataset Description

The dataset contains information about road accidents with multiple attributes.

### Key Columns

| Column                     | Description                             |
| -------------------------- | --------------------------------------- |
| Accident_Index             | Unique accident identifier              |
| Accident Date              | Date when accident occurred             |
| Day_of_Week                | Day of the accident                     |
| Junction_Control           | Type of junction control                |
| Junction_Detail            | Junction type                           |
| Accident_Severity          | Severity level (Slight, Serious, Fatal) |
| Latitude                   | Geographic latitude of accident         |
| Longitude                  | Geographic longitude of accident        |
| Light_Conditions           | Lighting condition during accident      |
| Weather_Conditions         | Weather at time of accident             |
| Road_Surface_Conditions    | Road surface condition                  |
| Road_Type                  | Type of road                            |
| Speed_limit                | Speed limit of the road                 |
| Number_of_Vehicles         | Number of vehicles involved             |
| Number_of_Casualties       | Number of casualties                    |
| Vehicle_Type               | Type of vehicle involved                |
| Urban_or_Rural_Area        | Accident location classification        |
| Police_Force               | Police department handling the case     |
| Local_Authority_(District) | District where accident occurred        |
| Carriageway_Hazards        | Road hazards present                    |
| Time                       | Time of accident                        |
